# ProveTok Figure Catalog

All 17 available figures for the paper and supplemental material.
This document records: what each figure shows, where the raw data lives, how to regenerate it, and suggested placement.

---

## Quick Regeneration Commands

```bash
# Regenerate all 10 redrawn figures (no GPU, ~10s)
python Scripts/figures_redraw_v2.py
# Output: outputs/redrawn_figures/*.{png,pdf}

# Regenerate old-style figures 1-3 + tables from 5K ablation data
python Scripts/generate_table2_and_figures.py --data_dir outputs/5k_ablation
# Output: outputs/paper_figures_5k/

# Regenerate case study figure (no GPU, reads trace data)
python Scripts/plot_case_study.py
# Output: outputs/redrawn_figures/fig_case_study.{png,pdf}

# Regenerate counterfactual perturbation data (no GPU, ~2min)
python Scripts/metric_sanity_check.py \
    --cases_dir outputs/stage0_5_5k/test/cases --max_cases 500
# Output: outputs/metric_sanity_check.json
```

### Fig S5. τ_IoU Critical Point (line chart)

| Item | Value |
|------|-------|
| **File** | `outputs/redrawn_figures/fig_tau_iou_critical_point.{pdf,png}` |
| **Script** | `Scripts/figures_redraw_v2.py` → `plot_tau_sweep()` (L101-137) |
| **Data source** | Hardcoded in script; original from `outputs/mediastinum_tau_sweep_50/` |

**What it shows:**
- τ_IoU sweep from 0.03 to 0.05. Mediastinum violation rate = 0% for τ≤0.04, jumps to 100% at τ=0.045.
- Overall violation rate: ~8.3% for τ≤0.04, rises to 12.8% at τ≥0.045.
- Annotated: τ_IoU = 0.04 selected as the critical point.

**Key data (hardcoded L102-104):**
```python
tau         = [0.03, 0.035, 0.04, 0.045, 0.05]
mediastinum = [0.0, 0.0, 0.0, 100.0, 100.0]
overall     = [8.27, 8.27, 8.27, 12.78, 12.78]
```

**Appendix story:** τ_IoU has a sharp phase transition at 0.04; below this threshold, mediastinum tokens are correctly included in routing; above, they are all filtered out, causing 100% violation for mediastinum sentences.
